# **App Name**: Mudrika Variya

## Core Features:

- Hero Section: Displays a brief introduction with title, subtitle, and links to 'View My Work' and 'Contact Me'.
- About Me Section: Provides a professional bio including experience, completed projects and hybrid app speciality.
- Skills Section: Showcases key skills with animated cards in categories: Mobile Development, Backend & APIs, Architecture, Performance Optimization, and Tools.
- Experience Timeline: Outlines professional experience in Quad Softtech, Emperic Infotech, and YohoSys.
- Projects Showcase: Displays app screenshot mockups and descriptions including key technologies and Play Store/App Store link.
- Services Offered: Highlights Flutter App Development, FlutterFlow Rapid Development, MVP Development, and API Integration.
- Contact Information: Presents contact options including email, LinkedIn, GitHub, and a contact form.

## Style Guidelines:

- Dark mode with primary color Navy Blue (#0F172A) for a modern aesthetic.
- Accent color: Soft Neon Purple (#BB86FC) for glows and highlights to enhance visual appeal.
- Use 'Poppins' as the primary font for headings and body text, offering a clean and modern feel.
- Mobile-first design with a single-page layout, optimized for 375px width and scaling properly to tablet (768px) and desktop (1200px+).
- Implement smooth scrolling, fade-in effects on scroll, slide-up animations, and button ripple effects for enhanced user experience.
